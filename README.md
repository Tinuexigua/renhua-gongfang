# 人话工坊

一键去除AI味，让机器文案变成自然人话。

## 在线使用

https://tinuexigua.github.io/renhua-gongfang/

无需注册，打开即用。

## 功能

- 粘贴AI生成的文案，一键改写成 **5 种风格**：小红书风、公众号风、知乎风、学术风、日常口语
- 免费使用，每日限量
- 支持一键复制改写结果
- 无需注册，打开网页就能用

## 适用场景

- 小红书博主：去掉AI味，让笔记读起来像真人写的
- 公众号运营者：把AI草稿改成有温度、有逻辑的文章
- 知乎答主：让回答条理更清晰、更像真人经验分享
- 学生/研究者：把口语化内容改成严谨学术风格，或反向操作
- 自媒体创作者：多平台内容风格快速适配

## 技术栈

- 纯前端 HTML + CSS + JavaScript
- DeepSeek API
- GitHub Pages 托管
